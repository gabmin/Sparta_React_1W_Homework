# 스파르타코딩 React 주특기 1주차 과제

### 기능 구현
Firebase의 Firebase를 이용하여 데이터를 관리한다.  
Redux를 이용하여 불러오기, 저장하기 기능을 구현하였다.  
버튼을 누르면 스크롤이 이동하도록 하였다.
